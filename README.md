### 🚀 Featured projects

- **[KairosEdge — Crypto Arbitrage Toolbox](https://github.com/Aziz-Zeghal/Crypto-Arbitrage-Toolbox)** · *personal · 2023 → today*<br>
  Modular Python toolbox automating Cash & Carry and funding-rate arbitrage across spot,
  futures and perpetual markets, built around a templater design pattern with live
  Bybit/Deribit API integration and spread monitoring.

- **[KuaiRec Video Recommender](https://github.com/Aziz-Zeghal/KuaiRec_video_recommender)** <br>
  Preference prediction on the KuaiRec dataset — distributed ALS and a Two-Tower neural
  network in Keras.

- **[Poetry Classification & Generation](https://github.com/EPITA-SCIA/NLP1-Poetry)** <br>
  Benchmark over a 60k+ multilingual poem dataset comparing 10+ methods, from classical ML
  to RNN-with-attention and GPT-2.

- **[Poisoned Teacher & CoT Distillation](https://github.com/EPITA-SCIA/NLP-2026)** <br>
  Benchmark of backdoor transfer through knowledge distillation, quantifying the
  Clean-Accuracy vs Attack-Success-Rate trade-off across model scales and poison ratios.

- **POSIX Shell in C** · *best team of the generation*<br>
  POSIX-compliant shell with a lexer-parser architecture supporting expansion, piping,
  subshells, redirections and built-in commands.
